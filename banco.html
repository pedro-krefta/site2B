Fundamentos de Banco de Dados

Professor: José Ricardo Maçaneiro

perguntas

Um SGBD (Sistema Gerenciador de Banco de Dados) é um software que atua como intermediário entre o usuário e os dados, permitindo criar, gerenciar, armazenar e acessar bancos de dados de forma segura e eficiente. Ele abstrai a complexidade técnica, oferecendo uma interface para manipulação, controle e recuperação dos dados, mantendo sua integridade e organização.

Os principais bancos de dados incluem Oracle Database, MySQL, Microsoft SQL Server e PostgreSQL, que são amplamente utilizados e dominam o mercado relacional. Outros sistemas populares incluem MongoDB (NoSQL), Redis (em memória) e Cassandra (NoSQL de colunas).

Um dicionário de dados é um registro de metadados que descreve a estrutura de um banco de dados, incluindo nomes de tabelas, atributos, tipos de dados, tamanhos, restrições e relacionamentos. Os tipos de dados de um banco de dados são os diferentes formatos de informação que podem ser armazenados, como números inteiros, textos, datas, valores booleanos, entre outros.

A modelagem de banco de dados é a criação de um modelo lógico para representar as informações e relacionamentos de um sistema. O MER (Modelo Entidade-Relacionamento) é o modelo conceitual que descreve os objetos (entidades) e como eles se conectam (relacionamentos), enquanto o DER (Diagrama Entidade-Relacionamento) é a representação gráfica e detalhada desse modelo, incluindo atributos, chaves e outros detalhes técnicos.
