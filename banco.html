O que é um SGBD? Tipos e Aplicações?

Um SGBD (Sistema Gerenciador de Banco de Dados) é um software que atua como intermediário entre o usuário e os dados, permitindo criar, gerenciar, armazenar e acessar bancos de dados de forma segura e eficiente.

Tipos de SGBD

  • Relacionais (SQL)
  • Não-relacionais (NoSQL)
  • Orientados a objetos
  • Em memória

Aplicações

  • Sistemas corporativos
  • Aplicativos web/mobile
  • Análise de dados
  • IoT e Big Data

O que é um dicionário de dados? Quais os tipos de dados de um banco de dados?

Dicionário de Dados

Registro de metadados que descreve a estrutura de um banco de dados, incluindo:

  • Nomes de tabelas e atributos
  • Tipos de dados e tamanhos
  • Restrições e relacionamentos
  • Descrições dos elementos

O que é uma modelagem de banco de dados? E o modelo DER? MER?

Modelagem de Banco de Dados

Criação de um modelo lógico para representar as informações e relacionamentos de um sistema, organizando dados de forma estruturada.

MER - Modelo Entidade-Relacionamento

Modelo conceitual que descreve:

  • Objetos (entidades)
  • Conexões (relacionamentos)
  • Atributos das entidades
  • Visão geral do sistema

DER - Diagrama Entidade-Relacionamento

Representação gráfica detalhada incluindo:

  • Atributos específicos
  • Chaves primárias e estrangeiras
  • Cardinalidades
  • Detalhes técnicos completos
