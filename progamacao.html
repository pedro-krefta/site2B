Linguagens de Programação

Conceitos fundamentais sobre desenvolvimento de software

O que são Linguagens de Programação? Fale sobre as seguintes linguagens: C++, Javascript, Java, Python, PHP.

Linguagens de programação são sistemas formais com regras e símbolos para criar softwares, atuando como uma ponte entre humanos e computadores.

C++: Usada para sistemas de alta performance e jogos, sendo uma linguagem orientada a objetos.

JavaScript: Essencial para desenvolvimento web front-end e back-end.

Java: Conhecida pela sua portabilidade ("escreva uma vez, execute em qualquer lugar") e robustez.

Python: Elogiada pela sintaxe clara e facilidade de aprendizado, usada em diversas áreas como ciência de dados e web.

PHP: Voltada principalmente para o desenvolvimento web no lado do servidor.

O que é uma linguagem de programação interpretada? E uma linguagem de programação compilada?

Linguagem Compilada: Traduz o código-fonte inteiro para código de máquina antes da execução, gerando um arquivo executável que roda diretamente no processador.

Linguagem Interpretada: Executa o código linha por linha em tempo real através de um interpretador, que faz a tradução durante a execução.

Diferenças: Linguagens compiladas tendem a ser mais rápidas e eficientes, enquanto as interpretadas oferecem maior portabilidade e flexibilidade.

O que é HTML e CSS

HTML e CSS são linguagens usadas para construir páginas web:

HTML (HyperText Markup Language): É a estrutura (o esqueleto) que define o conteúdo, como títulos, parágrafos e imagens.

CSS (Cascading Style Sheets): É a linguagem de estilo que cuida da aparência visual, definindo cores, fontes, layout e espaçamento.

Juntas, elas formam a base do desenvolvimento web front-end, que é a parte com a qual o usuário interage diretamente.

O que é uma IDE (ambiente de desenvolvimento de sistemas - exemplos: VSCode, IntelliJ, ...)

Uma IDE (Ambiente de Desenvolvimento Integrado) é um software que combina ferramentas para facilitar a escrita, compilação, teste e depuração de código em um único ambiente.

Exemplos comuns:

  • VS Code: Que pode funcionar como um IDE com plugins
  • IntelliJ IDEA: Projetado para aumentar a produtividade do desenvolvedor

As IDEs incluem recursos como auto-complete, destaques de sintaxe e gerenciamento de versão para melhorar a eficiência do desenvolvimento.
