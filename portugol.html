Algoritmos e Fluxogramas

Conceitos fundamentais de lógica de programação

O que são fluxogramas? Qual a sua simbologia básica?

Fluxogramas são diagramas visuais que representam a sequência de passos de um processo ou fluxo de trabalho. Eles usam uma variedade de símbolos padronizados e setas para mostrar as etapas, decisões e o fluxo de informações de maneira clara e concisa.

Simbologia básica:

  • Oval/Elipse: Início e fim do processo
  • Retângulo: Processo/ação a ser executada
  • Losango: Decisão (sim/não, verdadeiro/falso)
  • Paralelogramo: Entrada/saída de dados

O que são algoritmos? Onde são usados?

Um algoritmo é uma sequência de instruções bem definidas, normalmente usadas para resolver problemas de matemática específicos, executar tarefas, ou para realizar cálculos e equações.

Curiosidade histórica:

A origem da palavra remete a Al Khowarizmi, famoso matemático árabe do século IX, considerado o pai da álgebra.

O que são variáveis e constantes?

Variável

É um item de dados nomeado cujo valor pode ser alterado durante a execução de um programa, funcionando como uma caixa que pode guardar diferentes conteúdos.

Constante

É um item de dados nomeado com um valor predefinido que não pode ser modificado após sua definição, permanecendo fixo durante todo o programa.

Quais os tipos de dados mais usados em algoritmos (portugol)?

• Inteiro: Números sem casas decimais
• Real: Números com casas decimais
• Caractere: Um único caractere
• Cadeia: Sequências de caracteres (texto)
• Booleano: Valores lógicos (verdadeiro/falso)
• Vetor: Múltiplos valores do mesmo tipo

O que é o portugol? O que é o Portugol Studio?

Portugol

É uma linguagem de programação fictícia usada para ensinar lógica de forma simples, pois utiliza comandos em português.

Portugol Studio

É o programa onde esses algoritmos podem ser escritos, testados e executados, servindo como ambiente de aprendizado para iniciantes.
